# ETHOS · Sepsis

Tutor docente del reconocimiento y manejo de la **primera hora** de la sepsis del **adulto**
(cribado NEWS2/SIRS, hemocultivo y lactato, timing estratificado del antibiótico, fluidos,
meta de PAM y vasopresores, control de foco, SOFA pronóstico, traslado).

**Herramienta docente y de referencia — no es un dispositivo médico. El clínico decide.**
Estática, offline, sin datos de paciente.
